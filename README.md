# vikiruza — Школа танцев

Сайт-визитка школы танцев vikiruza.

**Сайт:** https://eudu.github.io/vikiruza/

## Стек

Чистый HTML + CSS, без фреймворков. Деплой через GitHub Pages.

## Структура

```
index.html   — одностраничник со всеми секциями
style.css    — стили, адаптив
images/      — фотографии (hero.jpg, about.jpg, gallery-1..5.jpg)
```

## Секции

- Hero
- О школе
- Направления
- Расписание
- Цены
- Галерея
- Контакты

## Локальный просмотр

Открыть `index.html` в браузере или запустить любой локальный сервер:

```bash
npx serve .
```
